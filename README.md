# nutrion_checker
A nutrition checker TUI using NutritionVerse-direct, a deep neural network trained to return nutrients information from food images.

# installation
This project is managed by [uv](https://docs.astral.sh/uv/), and uses [pytorch](https://pytorch.org/) to train the machine learning model. The TUI is built in go using [bubble tea](https://github.com/charmbracelet/bubbletea).

# nutrition-verse direct
[NutritionVerse-Direct](documents/2405.07814v1.pdf) is a joint research conducted by Waterloo and NRC in 2024 for the Aging in Place Challenge.
